# Azure VM PowerShell Deployment Guide

Quick deployment guide for Azure VMs using PowerShell.

## Quick Start

\`\`\`powershell
# Clone this repository
git clone https://github.com/YOUR_USERNAME/azure-vm-powershell-guide.git
cd azure-vm-powershell-guide

# Edit configuration
notepad scripts/Deploy-AzureVM.ps1

# Run deployment
.\scripts\Deploy-AzureVM.ps1
\`\`\`
