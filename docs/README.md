# Azure Ressources PowerShell Deployment Guide

Quick deployment guide for Azure Ressources using PowerShell.

## Quick Start

\`\`\`powershell
# Clone this repository
git clone https://github.com/Isamasm/azure-ressources-powershell
cd azure-ressources-powershell

# Edit configuration
notepad scripts/Deploy-Azure-ressources.ps1

# Run deployment
.\scripts\Deploy-Azure-ressources.ps1
\`\`\`
